Descubra os Mágicos Tipos de Dados em Python!

Em Python, existem tipos de dados mágicos para lidar com diferentes tipos de valores. Vamos conhecer alguns deles:

Números Inteiros (int)

Números inteiros são como varinhas mágicas para contar coisas! Eles não têm casas decimais. Exemplo:

numero_inteiro = 42

Números de Ponto Flutuante (float)

Números de ponto flutuante têm casas decimais e são ótimos para lidar com magia matemática! Exemplo:

numero_decimal = 3.14

Texto (String) (str)

Strings são como feitiços mágicos para lidar com palavras e frases! Elas são formadas por caracteres. Exemplo:

texto = "Olá, Python!"

Booleanos (bool)

Valores booleanos são como poções mágicas - eles podem ser verdadeiros ou falsos! Exemplo:

eh_verdadeiro = True eh_falso = False

Listas (list)

Listas são como sacolas mágicas que podem conter vários itens! Exemplo:

minha_lista = [1, 2, 3, 4, 5]

Dicionários (dict)

Dicionários são como pergaminhos mágicos com chaves e segredos! Exemplo:

meu_dicionario = {"nome": "Harry", "casa": "Gryffindor"}

Tuplas (tuple)

Tuplas são como encantamentos mágicos que protegem seus itens! Elas são imutáveis. Exemplo:

minha_tupla = (1, 2, 3)
