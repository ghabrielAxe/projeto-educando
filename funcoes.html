Descubra o Mundo das Funções em Python!

Em programação, uma função é como um feitiço mágico que realiza uma tarefa específica quando é chamado. Vamos aprender sobre funções com um exemplo divertido:

Definindo uma Função para Cumprimentar

def cumprimentar(nome):
    return "Olá, " + nome + "!"

Neste feitiço mágico chamado cumprimentar, nome é como um ingrediente especial. Quando você chama o feitiço com um nome, ele cria uma saudação mágica!

Chamando a Função

nome_do_usuario = "Harry"
saudacao_magica = cumprimentar(nome_do_usuario)
print(saudacao_magica)

Aqui, nós chamamos o feitiço cumprimentar com o nome "Harry" e a saudação mágica aparece: "Olá, Harry!".

Misturando Feitiços

def quadrado(numero):
    return numero * numero

numero = 5
resultado = quadrado(numero)
print("O quadrado de", numero, "é", resultado)

Aqui, usamos outro feitiço chamado quadrado para encontrar o quadrado de um número!
