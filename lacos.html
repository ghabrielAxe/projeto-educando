Explorando os Mágicos Laços de Repetição em Python

Laços de repetição são como encantamentos que permitem que você execute um bloco de código várias vezes. Eles são incrivelmente úteis quando você quer realizar uma tarefa repetidamente sem reescrever o mesmo código várias vezes. Vamos aprender sobre os poderosos laços de repetição em Python!

Loop While:Repetindo até a Condição ser Falsa

O loop while permite que você repita um feitiço enquanto uma condição for verdadeira. Enquanto a condição for mantida, o código dentro do loop será executado repetidamente!

quantidade_de_macas = 5
while quantidade_de_macas > 0:
    print("Você tem maçãs! Pode fazer uma torta mágica!")
    quantidade_de_macas = quantidade_de_macas - 1

Loop For: Repetindo com uma Lista de Ingredientes!

O loop for é como um feitiço que percorre uma lista de ingredientes mágicos. Ele executa um bloco de código para cada item na lista, permitindo que você trabalhe com cada elemento separadamente!

ingredientes = ["olhos de sapo", "asas de morcego", "pó de fada"]
for ingrediente in ingredientes:
    print("Adicionando", ingrediente, "ao caldeirão mágico!")

Break e Continue: Comandos de Controle para Laços

Break e Continue são feitiços especiais que alteram o comportamento dos loops. O break quebra o loop instantaneamente, enquanto o continue pula para a próxima iteração, ignorando o resto do código no loop.

for numero in range(1, 11):
    if numero == 5:
        print("Encontramos o número 5! Parando o feitiço.")
        break
    print("Este é o número", numero)
